# site_web_tuvalu
Projet de création de site web autour de l'archipel de Tuvalu.
Réalisé en janvier 2025 dans le cadre du projet de fin de semestre de la matière "Techno Web" en L1 d'Informatique.

Voici l'emplacement des indices sur la page de la devinette :
1 : Lorsque l'on clique sur le premier point d'interrogation de la phrase "Que direz vous de jouer à un jeu de devinette avec moi ?" on peut voir apparaitre le premier indice.
2 : Le deuxième indice se trouve en zoomant sur le chat.
3 : Le troisième se trouve en sélectionnant pendant quelques secondes les point d'interrogation sur la page.

Les autres points à observer sur la page sont les suivants :
Passé : Une carte qui permet de situer l'île en zoomant sur celle-ci en partant d'un point de vue spatial de la Terre.
	Un arbre hiérarchique dont les éléments se dévoile quand on place son curseur sur lui.
	Un dollar qui roule quand on a le curseur sur le contenu de la page.

Devinette : Une réponse s'affiche à l'envers quand on pointe son curseur sur celui-ci

Logo en haut à droite : Renvoie à l'accueil du site.

Images de fond : Les images de fond ont toujours un rapport avec la page en question est sont différentes sur chaque page afin de donner à chaque page une identité.

Le bandeau du haut reste toujours à la même place d'un point de vue du viewport.

Le chat sur lequel les messages s'ajoutent au fur et à mesure des pages
